# SeqTrack Composer

AI-powered music generator for the **Yamaha SEQTRAK** groovebox. Describe what you want to hear in plain language, get SEQTRAK-ready patterns, edit them in a visual step sequencer, and send everything to hardware over USB.

---

## Features

### AI Composition
Type a natural-language prompt like *"dark techno at 130 BPM"* or *"lo-fi hip hop with jazzy chords"* and receive complete multi-track patterns generated by Claude. The AI understands SEQTRAK's 11-channel architecture and produces musically coherent output across drums, synths, FM, and sampler tracks.

### Step Sequencer
11-track visual grid editor with click-to-toggle notes, velocity editing, per-pattern bar length (1-8 bars), swing control, and probability per step. Each track maps directly to a SEQTRAK channel.

### Sound Browser
Browse 2032+ factory presets organized by category (Kick, Snare, Clap, Bass, Pad, Lead, etc.). Select sounds via MIDI Bank Select + Program Change, with real-time CC parameter controls for cutoff, resonance, attack, decay, reverb send, delay send, and more.

### Device Connection
Auto-detect SEQTRAK via the Web MIDI API. Per-channel connectivity test sends a note on each of the 11 channels to verify the hardware link. SysEx is enabled for deep parameter control.

### MIDI Export
Download any project as a Standard MIDI File (.mid, Type 1) with all 11 tracks on their correct channels. Compatible with any DAW.

### Drum Presets
7 built-in drum styles -- basic_4x4, breakbeat, trap, house, techno, dnb, hiphop -- plus a Euclidean rhythm generator for algorithmic patterns.

### Project Save / Load
Projects persist in localStorage with full JSON export/import. Each project stores all track patterns, BPM, scale settings, scene data, and timestamps.

## Getting Started

### Prerequisites

- Node.js 20+
- A Chromium-based browser (Chrome or Edge) for Web MIDI support
- Yamaha SEQTRAK connected via USB-C (optional -- the app works without hardware)

### Setup

```bash
git clone <repo-url>
cd seqtrack-composer
npm install
```

Create `.env.local` in the project root:

```
ANTHROPIC_API_KEY=sk-ant-...
```

Start the dev server:

```bash
npm run dev
```

Open `http://localhost:3000` in Chrome. Connect your SEQTRAK via USB-C before or after loading the page -- the app auto-detects the device.

### Build

```bash
npm run build
npm run start
```

## SEQTRAK Channel Mapping

The SEQTRAK does **not** use General MIDI conventions. Each instrument has its own dedicated MIDI channel:

| Channel | Track | Type | Engine |
|---------|-------|------|--------|
| 1 | Kick | Drum | AWM2 Drum |
| 2 | Snare | Drum | AWM2 Drum |
| 3 | Clap | Drum | AWM2 Drum |
| 4 | Hat 1 (Closed) | Drum | AWM2 Drum |
| 5 | Hat 2 (Open) | Drum | AWM2 Drum |
| 6 | Perc 1 | Drum | AWM2 Drum |
| 7 | Perc 2 | Drum | AWM2 Drum |
| 8 | Synth 1 | Melodic | AWM2 Synth |
| 9 | Synth 2 | Melodic | AWM2 Synth |
| 10 | DX | Melodic | FM Synthesis |
| 11 | Sampler | Melodic/Perc | Sampler |

Drum tracks use pitch 60 (C3) for all hits. Melodic tracks use standard MIDI note numbers.

## Project Structure

```
src/
  app/
    (app)/              # Route group with sidebar + header layout
      compose/          # AI composition page
      editor/           # Step sequencer page
      sounds/           # Sound browser page
      device/           # MIDI device connection page
      projects/         # Project management page
    api/compose/        # POST endpoint for AI generation
  components/
    editor/             # Step grid and sequencer components
    layout/             # App shell, sidebar, header
    sound/              # Sound browser and CC parameter controls
    ui/                 # shadcn/ui primitives
  hooks/                # useMidiConnection, useSoundControl
  lib/
    ai/                 # System prompts, Zod schema for AI output
    midi/               # Domain logic: types, constants, patterns, export, SysEx
    webmidi/            # Device I/O: connection, sender, test utilities
  providers/            # ProjectProvider (React context for project state)
  types/                # Type declarations (midi-writer-js)
```
